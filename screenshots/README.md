# Screenshots Folder

## Instructions for Adding Screenshots

Please add the following screenshots to this folder before final submission:

### Required Screenshots:

1. **main-screen.png**
   - Full view of the main weather screen
   - Should show: temperature, weather icon, location, current conditions
   - Include: wind speed, humidity, pressure, air quality

2. **forecast-screen.png**
   - Scroll down to show the 5-day forecast section
   - Should display all 5 days with dates, temps, and weather icons

3. **search-feature.png**
   - Tap the search icon (ðŸ”) in the header
   - Show the search input box with keyboard visible
   - Optionally type a city name

4. **error-screen.png**
   - Turn off internet or enter invalid city like "zzzzz"
   - Show the error banner at the top of the screen
   - Or show "City not found" error message

### How to Take Screenshots:

**On Android (Emulator or Physical Device):**
- Press **Power + Volume Down** simultaneously
- Or use Android Studio screenshot tool

**On iOS (Simulator or Physical Device):**
- Simulator: Click **Device > Screenshot** in menu
- iPhone: Press **Side Button + Volume Up**
- Or use Xcode screenshot tool

**After capturing:**
1. Rename files to match the names above
2. Place them in this `screenshots/` folder
3. Verify they display in the README.md

---

**Note:** Screenshots must show the app running on a mobile device screen (not desktop browser view).
